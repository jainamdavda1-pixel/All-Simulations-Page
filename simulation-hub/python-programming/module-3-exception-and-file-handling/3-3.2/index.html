LEVEL 1Absolute Basics — File Opening Modes

What you learn: Before reading or writing, you must open() a file with the right mode. The mode controls what you can do and what happens to existing data. Getting the mode wrong can overwrite files instantly!

LEVEL 2Core Application — Reading Text Files

What you learn: Three ways to read: read() gets everything at once, readline() reads one line, readlines() returns a list of all lines. Always close files — or better, use with open().

LEVEL 3Intermediate — Writing & Closing Files

What you learn: write() writes a string (you must add \n yourself). writelines() writes a list. close() flushes the buffer to disk — without it, data may be lost. with handles this automatically.

LEVEL 4Advanced — Appending Text Files

What you learn: Mode "a" opens a file and positions the cursor at the end. New writes are added after existing content — nothing is overwritten. Mode "w" starts fresh every time. Understanding this difference is critical for log files.



Choose an operation above

LEVEL 5Expert — File Pointer · seek() · tell()

What you learn: Every open file has a position pointer. tell() returns the current position in bytes. seek(offset, whence) jumps to any position. This gives you random access to file content — essential for large files.
